= 利用vCenter Server的VMware vCenter外掛程式來保護資料NetApp Element
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以使用NetApp Element 適用於vCenter Server的VMware vCenter外掛程式、確保資料複本已建立並儲存在您需要的位置。為達成此目的、您可以建立及管理磁碟區和群組快照、設定快照排程、以及建立磁碟區和叢集配對關係、以便在遠端叢集之間進行複寫。
